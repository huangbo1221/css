同样可以达到目的，但是import的方式（css2.1的导入方式）有一个弊端！
当打开一个含有元素比较多的网页页面的时候，可能会先打开html的架子，然后再导入css的渲染。
所以经常会看到网页刚打开时比较单一，后续会变得好看。

link的方式就会全部导入后，再进行渲染。
